Loupe は、Linux 向けのモダンで高速な Spotlight 風ランチャーです。Electron と Node.js で構築されており、アプリケーションの起動、ファイルの検索、カスタムエイリアスなどをサポートしています。

## 特徴

-   **アプリケーション起動**: システムにインストールされたアプリケーション（`.desktop`ファイル）を検索して起動します。
-   **Flatpak サポート**: Flatpak アプリケーションの検索と起動に対応しています。
-   **ファイル検索**: ホームディレクトリやドキュメント、ダウンロードなどの主要なフォルダを素早く開けます。
-   **エイリアス機能**: よく使うアプリに短い名前を付けられます（例: `code` -> `Visual Studio Code`）。設定画面から GUI で管理可能です。
-   **プラグインアーキテクチャ**: 機能を簡単に拡張できるプラグインシステムを採用しています。
-   **モダンな UI**: ダークモードとブラー効果を取り入れた美しいデザイン。
-   **CLI ライクな挙動**: 起動すると即座に入力可能になり、フォーカスが外れると自動的に終了します。

## インストール

このリポジトリをクローンし、依存関係をインストールしてください。

```bash
git clone https://github.com/amania-jailbreak/loupe.git
cd loupe
npm install
```

## 使い方

アプリケーションを起動するには、以下のコマンドを実行します。

```bash
npm start
```

これをショートカットに登録して使用でも OK です

ランチャーが表示されたら、検索したいアプリ名やファイル名を入力してください。
矢印キーで選択し、`Enter`キーで実行します。

### エイリアスの設定

検索バーの右側にある「歯車アイコン」をクリックすると、設定画面が開きます。
ここで、独自のエイリアスを追加・編集・削除できます。

## 開発

### プロジェクト構成

-   `main.js`: Electron のメインプロセス。ウィンドウ管理やライフサイクルを制御します。
-   `renderer.js`: レンダラープロセス。UI の描画やイベントハンドリングを行います。
-   `pluginManager.js`: プラグインの読み込みと管理を行います。
-   `plugins/`: 各機能（Apps, Flatpak, Files など）の実装が含まれています。

### プラグインの追加

`plugins/` ディレクトリに新しい `.js` ファイルを作成することで、機能を拡張できます。プラグインは以下のメソッドをエクスポートする必要があります。

-   `search(query)`: 検索クエリを受け取り、結果の配列を返す非同期関数。
-   `execute(item)`: 選択されたアイテムを実行する関数。
